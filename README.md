# AI Website Builder 🤖

This is the full-stack project . The goal is to build a web application that takes natural language text prompts from a user and uses AI to generate clean, fully responsive website layouts.

To keep the codebase modular, scalable, and easy to maintain, I've structured the project with a distinct separation between the frontend user interface and the backend processing server.

## 📁 Project Structure

* **`/frontend`** – Contains the user interface, the prompt input dashboard, and the workspace where the generated web previews are rendered.
* **`/backend`** – Manages the server logic, processes API requests, and handles the communication layer with the generative AI models.
